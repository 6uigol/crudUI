# CRUD UI Hotel (SQL Server)

## 1) Configurar conexão
Edite o arquivo `config.js` na raiz para alterar servidor, usuário, senha e banco.

## 2) Criar tabelas
Execute o script `tabelas.sql` no SQL Server Management Studio.

## 3) Rodar projeto
```bash
npm install
npm start
```

Acesse:
- http://localhost:3000/index.html
